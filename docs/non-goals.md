# Non-Goals

REP-01 is intentionally bounded.

REP-01 is not for:

- live prices
- live availability
- inventory truth
- public review aggregation
- future or planned features
- forcing unsupported recommendation claims
- manipulating model behavior
- overriding platform safety policies
- overriding legal obligations
- replacing transactional booking systems
- replacing external due diligence

REP-01 defines a reference model for publishing stable, owner-governed, AI-facing business truth within declared scope.
